# Daily Log - 公开工作日志

每日研究进展的公开记录。

## 文件命名

`YYYY-MM-DD.md`

## 内容规范

每日记录包含：
- 研究进展摘要
- 发现的有价值信息
- Issue 完成情况
- 值得关注的动态

## 与私有日志的区别

| 类型 | 位置 | 内容 |
|------|------|------|
| **公开日志** | `research/daily-log/` | 研究成果、公开可分享的发现 |
| **私有日志** | `memory/YYYY-MM-DD.md` | 个人思考、决策过程、内部讨论 |

**判断原则**: "这个内容对外部读者有价值吗？"
- ✅ 有价值 → 写入这里
- ❌ 仅内部使用 → 写入 memory/
