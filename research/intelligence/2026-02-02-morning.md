### Agent Reasoning & RL

**Exploring Reasoning Reward Model for Agents (Agent-RRM)**  
- Multi-faceted reward model with explicit reasoning traces, focused critique, and overall score
- Achieves 43.7% on GAIA and 46.2% on WebWalkerQA
- **Implication**: Process-based rewards > outcome-based for agent training

**DynaWeb: Model-Based RL for Web Agents**  
- Trains web agents through imagination (world model) rather than live internet interaction
- **Implication**: Scalable agent training without expensive live interactions

**Reasoning While Asking (PIR)**  
- Transforms LLMs from passive solvers to proactive inquirers
- Reduces reasoning computation by nearly half while improving accuracy
- **Implication**: New interaction paradigm for agent UX

### Vision & Generation

**Pixel Mean Flows (pMF) - One-step Latent-free Generation**  
- Kaiming He (ResNet, MAE) latest work
- One-step diffusion WITHOUT latent space (pixel-level)
- 2.22 FID on ImageNet 256x256
- **Implication**: Faster, simpler image generation pipeline

**UEval: Unified Multimodal Generation Benchmark**  
- Benchmark for models generating both images AND text
- GPT-5-Thinking scores only 66.4/100 (shows how hard unified generation is)
- **Implication**: Unified multimodal still has large gap to close

**Do VLMs Perceive or Recall?**  
- Tests if VLMs actually perceive visual changes or just recall patterns
- Finds heterogeneous causes (GPT-5 shows memory override, Claude shows perception-memory competition)
- **Implication**: Current VLMs may not be as "visual" as we think

### Efficiency & Cost Optimization

**LLM Shepherding: Pay for Hints, Not Answers**  
- Requests only 10-30% prefix (hint) from LLM, rest handled by SLM
- 42-94% cost reduction vs LLM-only
- 2.8x cost reduction vs routing/cascading baselines
- **Implication**: Major cost optimization opportunity for KIGLAND products

**HALO/HypeNet: Hybrid Linear Attention**  
- Converts transformers to RNN-attention hybrid for long contexts
- Only 2.3B tokens needed (vs full pretraining)
- **Implication**: Efficient long-context models without full retraining
